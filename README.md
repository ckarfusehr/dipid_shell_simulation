# dipid_shell_simulation

This repository contains the code used for capsid simualtions in the publication: Karfusehr, C., Eder, M., Yang, H.Y. et al. Self-assembled cell-scale containers made from DNA origami membranes. Nat. Mater. (2025). https://doi.org/10.1038/s41563-025-02418-0. The main repository can be found [here](https://github.com/ckarfusehr/DNA_origami_membranes.git). Additional Data is available on figshare.
